Chapter 8: Transforms and Transitions

We can translate, rotate, scale and skew any element on the page by using (transform) and CSS3 property. We can manipulate an element's appearance using transform functions.

With the transform fuctions we can move element left, right, up or down. Similar to the behavior (position: relative;)

(translate (x,y) moves an element x from the left and y from the top)

  • transform: translate (45px, -45px);
  • translatex or translatey to move and element vertically or horizontally
  • Transform don't work on inline elements

    • Scaling: (scale (x,y)) a function that scales and element by the defined factor horizontally then vertically

  • scaleX(x) or scaleY(y) will scale only the horizontal dimensions or only the vertical dimensions respectively

    • Rotation: rotate() this functions rotates an element around the point of origin by a specified angle value

  • .ad-ad2 h1:hover span { color: #484848; transform: rotate(10deg) translateX(40px) scale(1.5); }

    • Skew: skew (x,y) whit this function we can specifie a skew along the x and y exes

  • transform: skew(15deg, 4deg);

    • Transitions: allow the values of CSS properties to change over time, essentially providing simple animations

  • transition-duration: sets how long the transition will take
  • transition-duration: 0.2s;
  • transition-timing-function: control the pace of the transition in even more granular detail
  • Most common timming functions: ease, linear, ease-in, ease-out or ease-in-out
  • transition delay: we can introduce a delay before the transitio begins

    • Animations: this will help us to control each step of an animation via keyframes

    Chapter 12: Canvas, SVG, and Drag and Drop

    Thanks to Canvas API we can draw anything we can imagine, all through JavaScript

  • The first step to use canvas is to add (canvas) element to the page
  • We need to set the width and height here and not in the CSS

    • SVG: (Scalable Vector Graphics) file format that allows us to describe vector graphics using XML

  • XML stands for eXtensible Markup Language it's a markup metalanguage
  • To draw we can use (svg)
  • We can use filters and svg to make it black and white

    • Drag on Drop: this API allows us to specify that certain elements are draggable, and then specify what should happen when these draggable elements are dragged over or dropped onto other elements on the page
