Chapter 9: The Window Object

Browser Object Model (BOM) is a collection of properties and methods and contain information about the browser and computer screen

In general, you should refer to global variables without using the window object; it’s less typing and your code will be more portable between environments. An exception is if you need to check whether a global variable has been defined

  • Global variables are created without using the const, let or var keywords
  • Most browsers allow users to disable any dialogs from repeatedly appering, meaning they are not a feature to be relied upon
  • window.location.href: return full URL

    • With window.history we can access information about any previously visited pages in the current browser session

  • window.history.length : shows how many pages have been visited before arriving at the current page
  • window.open() : we can open a new window
  • window.screen: information about the screen the browser is displayed on, if we use (height) and (width) we can know the height and width of the screen in pixels

    • Each window object contains a document object. The document object contains a few other methods that are worth looking at

  • write(): simply writes a string of text to the page

    • A cookie is a small piece of text that the websites you visit send to your browser. Cookies allow sites to remember information about your visit, which can make sites easier to revisit and more useful to you

  • To create a cookie we need to assign it to JavaScript's, using the document.cookie
  • document.cookie = 'name=Superman'

    • Timing functions: setTimeout(), setInterval()

    Chapter 14: HTML5 APIs

    HTML5 is the lastest version of the Hypertext Markup Language used to creat web pages

  • data: this is a way of embedding data in a web page and we use custom attributes that are ignored by the browser

    • We can decide these attributes but we need to follow some rules

  • Star with data
  • Contain only lowercase letters, numbers, hyphens, dots, colons or underscores
  • Include an optional string value

    • APIs that are popular and supported in most moder browsers

  • HTML 5 Web Storage
  • Geolocation

    • The list of APIs is constantly growing, and includes APIs for accessing a device’s camera, uploading files, accessing the battery status, handling push notifications, building drag-and-drop functionality, creating 3D effects with WebGL, and many more!
